tukanoid_styling library Null safety

My package for helping me and people who work with me on flutter project to style applications in a way that doesn't require setting all the styling parameters inside of the initialization of each widget but rather in a separate objects that can then turn into those widgets

Classes

ContainerImageStyling
Class used for styling both Image and Container widgets, since both can use ImageProvider as background image source (Container) or as just image source (Image)
ContainerStyling
Class for styling container-specific attributes, used mostly in ContainerImageStyling
DividerStyling
Class made for styling the Divider widget
EdgeInsetsStyling
Class for making EdgeInsets be dependent on screen size percentage rather than set pixel value
IconStyling
Class made for styling the Icon widget
Position
Class that is used for Positioned widget to define the position of a Widget in a Stack
PositionedStyling
Class for styling the Position class by utilizing percentage of the screen size rather than set pixel size
Styling
TappableIcon
Similar to IconButton but with different animation color animation approach
TappableText
Class that makes Text or AutoSizeText widget tappable by using InkResponse as a wrapper for gesture detection
TextFieldStyling
Class for stying a TextField
TextStyling
Class used for styling a Text or AutoSizeText widget
VerticalDividerStyling
Class made for styling the VerticalDivider widget

Extensions

ListExtension
List extension helper functions
SizeExtension
Size extension helper functions
StringExtensio
String extension helper functions
WidgetExtension
Widget extension helper functions
